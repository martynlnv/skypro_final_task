Git

  • git init - Создает новый локальный репозиторий с заданным именем.
  • git clone - Скачивает репозиторий вместе со всей его историей изменений.
  • git status - Перечисляет все новые или измененные файлы, которые нуждаются в фиксации.
  • git add [файл] - Индексирует указанный файл для последующего коммита .
  • git push - Выгрузка сделанных изменений в удаленный репозиторий.
